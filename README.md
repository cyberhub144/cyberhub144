**Tech Stack:** `Python` · `Scapy` · `SQLite` · `Bash`

**The challenge:** Detect and alert on suspicious network activity in real time without commercial tools.

**What I built:**
- 🔍 Packet sniffer that monitors live traffic and flags anomalies using rule-based detection
- 📊 SQLite logging system for storing and querying captured threat events
